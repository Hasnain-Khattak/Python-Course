# Mini Project: Word Frequency Analyzer

Create a program that:

Asks the user to input a sentence.

Uses a list comprehension to create a list of words from the sentence.

Counts the frequency of each word using a dictionary (where keys are words and values are the counts).

Uses a list comprehension to filter out words that appear only once.

Prints the resulting list of words that appear more than once.
